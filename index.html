About

Welcome to Emerald MC! Emerald MC is a small vanilla Minecraft server with support for both bedrock and java editions of Minecraft. Emerald MC is not a public server and you will not be able to join if you aren't on the whitelist. This website has been made to make it easy to find any information you want about Emerald MC. If you have any suggestions for this, you can contact me via the Emerald MC group chat.

Server Rules

Emerald MC is a friendly server with no fighting, griefing, or other things like that. Stealing and cheating / hacking is also against the rules and is strictly prohibited, breaking any of these rules will result in a one day ban. Repeatedly breaking these rules will increase the ban time more.

Allowed Mods

Any mods that don't affect the gameplay too much are allowed, mods like Sodium and any other performance boosting mods are allowed, along with other mods that change how the game looks. Any mods that give you cheats or unfair advantages are not allowed. This does not include fullbright or low on fire. If you have any questions about modding, you can ask me in the group chat.
